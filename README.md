## Задание

Ваша задача — написать приложение для клуба путешествий с возможностью поиска, фильтрации по цене и выбора самого дешёвого варианта.

При нажатии на кнопку **"Показать всё"** должен отправляться запрос на бэкенд, а полученные данные отображаться в контейнере ниже в виде карточек путешествий.

После ввода данных в форму и нажатия на кнопку должна выполняться фильтрация путешествий по цене.

Приложение взаимодействует с сервером по API и отображает данные в виде карточек.

## Запуск и сборка приложения

Для запуска используйте команду:

```bash
make run # запускает сервер и сборку

```

## Задача 1

В файле **src/app.js** напишите и экспортируйте по умолчанию функцию, которая создаёт форму со следующими элементами, используя нативные методы JavaScript:

- Кнопка **"Показать всё"**.
- Фильтр с двумя полями: **"Цена от"** и **"Цена до"** (числовой ввод).
- Кнопка **"Применить фильтр"**.

Форма должна быть добавлена в контейнер с классом **search-form-container**. После выполнения JavaScript DOM-элемент с формой должен выглядеть следующим образом:

### Форма

```html
 <form class="search-form">
        <button type="submit" class="show-all-button">
          Показать всё
        </button>
        <label for="price_min">Цена от:</label>
        <input
          type="number"
          name="price_min"
          id="price_min"
          class="price-input"
          placeholder="Например, 10 000"
          min="0"
        />
        <label for="price_max">Цена до:</label>
        <input
          type="number"
          name="price_max"
          id="price_max"
          class="price-input"
          placeholder="Например, 100 000"
          min="0"
        />
        <button type="button" class="search-button">
          Применить фильтр
        </button>
      </form>
```

## Задача 2

Напишите функцию, которая при клике на кнопку **"Показать всё"** отправляет запрос на сервер и, в случае успешной отправки, выводит товары в виде карточек в контейнере с классом **travels**

### Условия

- При клике на кнопку **"Показать всё"**, должен совершаться GET запрос по адресу **/travels**.
Ответ приходит в виде json. Пример объекта путешествия:

```json
  [{
    id: 7,
    title: 'По следам Золотого кольца',
    description: 'Культурный маршрут по древним городам России.',
    price: 35000,
    location: 'Россия',
    travelType: 'экскурсионный',
    images: ['/assets/rostov.jpg'],
  }]
  ```

- В случае успешного ответа все путушествия должны быть выведены в контейнере с классом **travels** в следующем виде для каждой карточки:

```html
<div class="card">
  <img src="/assets/rostov.jpg" alt="По следам Золотого кольца" />
  <div class="card-body">
    <h5 class="card-title">По следам Золотого кольца</h5>
    <p class="card-text">Культурный маршрут по древним городам России.</p>
    <h3 class="card-text">От 350000 ₽</h3>
  </div>
</div>
```

- В случае ошибки в контейнер с классом  **travels** нужно добавить тег **p** с текстом **Ошибка загрузки данных**.

```html
 <p class="error">Ошибка загрузки данных</p>
 ```

## Задача 3

Добавьте в функцию логику фильтрации по цене. Повторный запрос на бэкенд отпарвлять не нужно. Общее количество путешествий, подходящих под условие должно отображаться в специальном контейнере с классом **travels-number**.

```html
<div class="travels-number">Найдено: {{количество путешествий}}</div>
```

В контенейнере с классом **travels** должны отображаться только подходящие пушествия.
